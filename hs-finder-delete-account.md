## HS Finder 계정 삭제 방법

### 앱 내에서 삭제 (권장)

1. HS Finder 앱 실행
2. 우측 하단 **설정** 탭 이동
3. **계정 탈퇴** 버튼 클릭
4. 확인 후 계정 및 데이터 삭제 완료

### 이메일로 요청

앱 접근이 불가능한 경우, 아래 이메일로 계정 삭제를 요청하세요.

- 이메일: support@suimworks.com
- 제목: `[HS Finder] 계정 삭제 요청`
- 내용: 가입 시 사용한 Google 계정 이메일 주소

요청 후 **7일 이내** 처리됩니다.
